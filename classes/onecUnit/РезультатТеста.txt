
Перем мКолЗапущено;
Перем мКолПровалено;

Процедура Конструктор()

	мКолПровалено = 0;

	мКолЗапущено = мКолЗапущено + 1;
КонецПроцедуры // ЗарегистрироватьЗапуск

Процедура ЗарегистрироватьПровал() Экспорт
	мКолПровалено = мКолПровалено + 1;
КонецПроцедуры // ЗарегистрироватьПровал

Функция Сводка() Экспорт
	Возврат "запущено: " + мКолЗапущено + "; провалено: " + мКолПровалено;
КонецФункции // Сводка

	Возврат мКолЗапущено;

Функция ПолучитьКолПровалено() Экспорт
